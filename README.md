### GUI

- 左上：显示代码段 可通过文件读取或命令窗口输入 需要加行号
- 右上：程序执行后的结果
- 下：每行代码的语法树 代码有问题的会显示错误

### 命令输入窗口作用： 

- 控制台指令
  - RUN
  - LOAD
  - CLEAR
  - HELP
  - QUIT

- 行号 代码 加入到程序中
- 代码 直接执行
- 代码中INPUT命令的交互
